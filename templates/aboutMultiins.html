Multi-Institute Support

🏢 Independent Institutes

Each registered institute operates in its own secured environment. No institute can access another’s students, subjects, or attendance data.

👨‍💼 Separate Institute Managers

Every institute is managed by its own manager account. Managers can only create subjects, add students, and view logs for their own institute.

🔑 Isolated Login System

Login pages are separated for students and managers. Users select their institute during login/registration so that authentication stays scoped.

📊 Scalable Database Design

Subjects, students, attendance logs, and settings are linked via institute IDs. This makes the system scalable across schools, colleges, and coaching centers.

🛡️ Secure Data Separation

Institute-specific data is strictly separated in the backend using role-based access and filtering. Even superusers require permissions to view institute data.
